HashMap

基础知识

hashCode() 方法

hashCode()会返回一个int类型的数值（int占4个字节32位）

获取下标

index = hash & (table.length - 1)，效果如 hash%table.length

Tip: table的长度是 2 的次幂原因之一，%本身能有效的减少hash冲突，在长度是 2 的次幂情况下能够将%转换成&，提高效率

主要方法

resize()

Tip: table的长度是 2 的次幂原因之二，避免重新hash,直接根据二进制的特性以及与运算来重新布局元素，提高效率

其中链表节点的重新布局

eg: 扩容前桶长度是 16，Node结构的元素a在旧桶的位置为index

  • 依托二进制以及&运算，且length是 2 的次幂，并且在两倍扩容的特性下，对于元素a在新桶的位置，要么在index，要么在[index+16]
  • 原Node结构的元素被分成两个Node，所以此处定义了四个指针:
    • loHead，loTail分别指向原位置的收尾
    • hiHead，hiTail分别指向新位置的收尾
  • 最后将新桶的index指向loHead，[index+16]指向hiHead

putVal()

  1. 如果table是空的，调用resize()来都到初始化的效果
  2. 根据hash找下标，如果该位置没有元素，直接放入
  3. 如果要放入的 key 已经存在，根据onlyIfAbsent决定是否覆盖原值
  4. 要放入的位置是红黑树结构，按照红黑树去放置元素
  5. 要放置的位置是Node结构，在链表上进行指针后移，如果key 已经存在，根据onlyIfAbsent决定是否覆盖原值，key 不存在，就将元素放在链表的尾部
  6. 在 5 的基础上，在加入了新的元素后，如果链表长度达到了 8，就进行树化，树化的另一个要求是桶的长度达到 64
  7. 当前桶长度如果达到了扩容阈值，进行扩容操作

getNode()

  1. 根据 hash 查找下标
  2. 判断下标的第一个元素的 key 是都跟要查找的相同，相同直接返回
  3. 当前元素是树结构，在树上查找并返回
  4. 当前元素是链表结构，从链表的第二个开始依次往后遍历

IntelliJ IDEA 2021.3激活

IntelliJ IDEA 2021.3激活

tip

此方法适合 jetbrains 2021.3全家桶，比如GataGrip、CLion等

先决条件

  • 移步https://www.jetbrains.com/idea/download/other.html 下载2021.3版本

  • 获取fineagent.jar文件: call me !!!

  • 激活码

windows

  1. 将jar文件放在一个英文目录，如 D:\plugins\jetbrains\fineagent.jar
  2. 安装下载的IntelliJ IDEA
  3. 安装完成打开，登陆账号（新下载的需要登陆账号），没有账号就注册一个，登陆成功继续试用
  4. 完成步骤3，新建一个java项目，点击 help ->Edit Custom VMOption…
  5. 输入-javaagent:${filepath}如:-javaagent:D:\plugins\jetbrains\fineagent.jar，保存
  6. 重启IntelliJ IDEA，选择Activation Code,输入激活码，点击Activate
  7. 点击help -> about 查看有效期限

macos

  1. 将jar文件放在一个英文目录，如 /usr/local/fineagent.jar
  2. 安装下载的IntelliJ IDEA
  3. 安装完成打开，登陆账号（新下载的需要登陆账号），没有账号就注册一个，登陆成功继续试用
  4. 完成步骤3，关闭软件，进入/Applications/IntelliJ IDEA.app/Contents/bin/，打开idea.vmoptions文件
  5. 输入-javaagent:${filepath}如:-javaagent:/usr/local/fineagent.jar，保存
  6. 重启IntelliJ IDEA，选择Activation Code,输入激活码，点击Activate
  7. 点击IntelliJ IDEA -> About IntelliJ IDEA 查看有效期限
