IntelliJ IDEA 2021.3激活

IntelliJ IDEA 2021.3激活

tip

此方法适合 jetbrains 2021.3全家桶，比如GataGrip、CLion等

先决条件

  • 移步https://www.jetbrains.com/idea/download/other.html 下载2021.3版本

  • 获取fineagent.jar文件: call me !!!

  • 激活码

windows

  1. 将jar文件放在一个英文目录，如 D:\plugins\jetbrains\fineagent.jar
  2. 安装下载的IntelliJ IDEA
  3. 安装完成打开，登陆账号（新下载的需要登陆账号），没有账号就注册一个，登陆成功继续试用
  4. 完成步骤3，新建一个java项目，点击 help ->Edit Custom VMOption…
  5. 输入-javaagent:${filepath}如:-javaagent:D:\plugins\jetbrains\fineagent.jar，保存
  6. 重启IntelliJ IDEA，选择Activation Code,输入激活码，点击Activate
  7. 点击help -> about 查看有效期限

macos

  1. 将jar文件放在一个英文目录，如 /usr/local/fineagent.jar
  2. 安装下载的IntelliJ IDEA
  3. 安装完成打开，登陆账号（新下载的需要登陆账号），没有账号就注册一个，登陆成功继续试用
  4. 完成步骤3，关闭软件，进入/Applications/IntelliJ IDEA.app/Contents/bin/，打开idea.vmoptions文件
  5. 输入-javaagent:${filepath}如:-javaagent:/usr/local/fineagent.jar，保存
  6. 重启IntelliJ IDEA，选择Activation Code,输入激活码，点击Activate
  7. 点击IntelliJ IDEA -> About IntelliJ IDEA 查看有效期限
